# Notes

## Typography

| Level | Name    | HTML Role / Notes                                                             |
| ----- | ------- | ----------------------------------------------------------------------------- |
| 1     | Title   | `<h1>` — Main page or article heading                                         |
| 2     | Article | `<h2>` — or `<article>` Self-contained content; may include multiple sections |
| 3     | Section | `<h3>` — or `<section>` Logical division inside an article.                   |
| 4     | Topic   | `<h4>` — Heading inside a `<section>`                                         |
| 5     | Detail  | `<h5>` — Finer breakdown under a topic                                        |
| 6     | Unit    | `<h6>` — Smallest content unit;                                               |
