CSS Custom Properties

Override any variable in :root to customize the framework:

:root {
  --font-family: "Inter", sans-serif;
  --border-color: #ccc;
  --shadow-color: 50, 50, 50;
  --br-2: 0.5rem;
  --sp-4: 20px;
}
