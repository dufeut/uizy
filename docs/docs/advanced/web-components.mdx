# Web Components

Uizy provides a set of custom web components that make building application layouts a breeze. These components work together to create responsive, animated app shells with headers, footers, sidebars, and drawers.

## Installation

To use the web components, import the Uizy JavaScript module:

```javascript
import uizy from "@dufeut/uizy";
```

## Quick Start

Here's a complete example showing all the web components working together:

```html
<uizy-app id="app">
    <!-- System Bar -->
    <uizy-system-bar>
        System bar
    </uizy-system-bar>

    <!-- Header -->
    <uizy-header shadow="12">
        <span></span>
        Header
        <span></span>
    </uizy-header>

    <!-- Left Drawers -->
    <uizy-drawer open shadow="6" clip-top clip-bottom>
        Left Sidebar
    </uizy-drawer>
    <uizy-drawer open mini shadow="6" clip-system clip-bottom>
        Left Mini
    </uizy-drawer>

    <!-- Right Drawers -->
    <uizy-drawer right open shadow="6" clip-top clip-bottom>
        Right Sidebar
    </uizy-drawer>
    <uizy-drawer right open mini shadow="6" clip-system clip-bottom>
        Right Mini
    </uizy-drawer>

    <!-- Overlay -->
    <uizy-overlay clip-top clip-bottom></uizy-overlay>

    <!-- Main Content -->
    <uizy-main clip-top clip-bottom clip-left clip-right>
        <p>Main content here</p>
    </uizy-main>

    <!-- Footer -->
    <uizy-footer shadow="12">
        Footer
    </uizy-footer>
</uizy-app>
```

## Components Reference

### `<uizy-app>`

The root container that manages all layout components. Required for the JavaScript API to work.

| Attribute | Description |
|-----------|-------------|
| `id` | Required for accessing via `uizy.get()` |

```html
<uizy-app id="app">
    <!-- All other components go here -->
</uizy-app>
```

---

### `<uizy-system-bar>`

An optional top bar for system status, notifications, or branding. Sits at the very top of the viewport.

| Attribute | Description |
|-----------|-------------|
| `shadow` | Shadow intensity (1-24) |

```html
<uizy-system-bar>
    Status: Online | v1.0.0
</uizy-system-bar>
```

:::tip
Configure the height with `--uizy-system-bar-height`. Set to `0px` if not using a system bar.
:::

---

### `<uizy-header>`

The main application header, positioned below the system bar.

| Attribute | Description |
|-----------|-------------|
| `shadow` | Shadow intensity (1-24) |
| `clip-*` | Clipping attributes (see below) |

```html
<uizy-header shadow="12">
    <button>Menu</button>
    <h1>My App</h1>
    <button>Settings</button>
</uizy-header>
```

:::info
Uses flexbox with `space-between` by default. Configure height with `--uizy-header-height`.
:::

---

### `<uizy-footer>`

Fixed footer at the bottom of the viewport.

| Attribute | Description |
|-----------|-------------|
| `shadow` | Shadow intensity (1-24) |
| `clip-*` | Clipping attributes |

```html
<uizy-footer shadow="12">
    © 2024 My App
</uizy-footer>
```

---

### `<uizy-drawer>`

Animated slide-in panels for navigation, sidebars, or contextual content.

| Attribute | Description |
|-----------|-------------|
| `right` | Position on right side (default: left) |
| `open` | Shows the drawer |
| `mini` | Collapsed/mini state |
| `shadow` | Shadow intensity (1-24) |
| `clip-top` | Clip below header |
| `clip-bottom` | Clip above footer |
| `clip-system` | Clip below system bar only |

```html
<!-- Left sidebar -->
<uizy-drawer open shadow="6" clip-top clip-bottom>
    Main Navigation
</uizy-drawer>

<!-- Right sidebar -->
<uizy-drawer right open shadow="6" clip-top clip-bottom>
    Details Panel
</uizy-drawer>

<!-- Mini sidebar (collapsed) -->
<uizy-drawer open mini shadow="6" clip-system clip-bottom>
    Icons Only
</uizy-drawer>
```

:::tip
You can have multiple drawers on each side - typically a full drawer and a mini drawer.
:::

---

### `<uizy-overlay>`

Semi-transparent backdrop for modals, drawers, and focus states.

| Attribute | Description |
|-----------|-------------|
| `clip-top` | Position below header |
| `clip-bottom` | Position above footer |

```html
<uizy-overlay clip-top clip-bottom></uizy-overlay>
```

---

### `<uizy-main>`

The main content area with full viewport height.

| Attribute | Description |
|-----------|-------------|
| `clip-top` | Add margin below header |
| `clip-bottom` | Add margin above footer |
| `clip-left` | Add margin for left sidebar |
| `clip-right` | Add margin for right sidebar |

```html
<uizy-main clip-top clip-bottom clip-left clip-right>
    <div class="pa-4">
        <h1>Welcome</h1>
        <p>Your content here</p>
    </div>
</uizy-main>
```

---

## Clipping Attributes

Clip attributes adjust positioning to account for other layout elements:

| Attribute | Description |
|-----------|-------------|
| `clip-top` | Position below system bar + header |
| `clip-bottom` | Position above footer |
| `clip-left` | Add left margin for sidebar |
| `clip-right` | Add right margin for sidebar |
| `clip-system` | Position below system bar only |

:::info
Horizontal clips (`clip-left`, `clip-right`) automatically reset on mobile breakpoints.
:::

## JavaScript API

### Initialization

Configure layout dimensions and breakpoints:

```javascript
import uizy from "@dufeut/uizy";

uizy.layout({
    overlay: {
        opacity: 0.45,
        color: "black",
    },
    layout: {
        system: 24,      // System bar height (px)
        header: 56,      // Header height (px)
        footer: 48,      // Footer height (px)
        left: 240,       // Left drawer width (px)
        right: 240,      // Right drawer width (px)
        leftMini: 64,    // Left mini drawer width (px)
        rightMini: 64,   // Right mini drawer width (px)
        drawerSpeed: 0.2 // Animation speed (seconds)
    },
    breakpoint: {
        name: "md",      // Current breakpoint name
        main: true,      // Adjust main content
        header: true,    // Adjust header
        top: true,       // Adjust top clipping
        bottom: false,   // Adjust bottom clipping
        left: true,      // Adjust left drawer clipping
        right: true,     // Adjust right drawer clipping
    },
});
```

### Running Code After DOM Ready

```javascript
uizy.init(() => {
    // Code here runs after DOMContentLoaded
    console.log("Uizy is ready!");
});
```

### Accessing Components

```javascript
uizy.init(() => {
    const app = uizy.get("app"); // Get by id
    if (!app) return;

    // Use the app instance...
});
```

### Controlling Components with Actions

The `action()` method lets you control drawers and overlays programmatically:

```javascript
uizy.init(() => {
    const app = uizy.get("app");
    if (!app) return;

    // Close overlay
    app.action("overlay", ({ set }) => set(false));

    // Toggle left drawer
    app.action("left", ({ set }) => set());

    // Toggle left mini drawer
    app.action("left.mini", ({ set }) => set());

    // Toggle right drawer
    app.action("right", ({ set }) => set());

    // Toggle right mini drawer
    app.action("right.mini", ({ set }) => set());

    // Set overlay to auto (follows drawer state)
    app.action("overlay", ({ set }) => set(null));
});
```

#### Action Targets

| Target | Description |
|--------|-------------|
| `overlay` | Controls the overlay visibility |
| `left` | Controls the left drawer |
| `left.mini` | Controls the left mini drawer |
| `right` | Controls the right drawer |
| `right.mini` | Controls the right mini drawer |

#### Set Function

The `set()` function accepts:

| Value | Description |
|-------|-------------|
| `true` | Open/show the element |
| `false` | Close/hide the element |
| `undefined` or no arg | Toggle the current state |
| `null` | Auto mode (overlay follows drawer state) |

## CSS Variables Reference

Customize the layout via CSS custom properties:

```css
:root {
    /* System Bar */
    --uizy-system-bar-height: 24px;
    --uizy-system-bar-layer: 40;

    /* Header */
    --uizy-header-height: 56px;
    --uizy-header-layer: 20;

    /* Footer */
    --uizy-footer-height: 48px;
    --uizy-footer-layer: 20;

    /* Left Drawer */
    --uizy-left-width: 240px;
    --uizy-left-mini-width: 64px;
    --uizy-left-layer: 20;

    /* Right Drawer */
    --uizy-right-width: 240px;
    --uizy-right-mini-width: 64px;
    --uizy-right-layer: 20;

    /* Drawer Animation */
    --uizy-drawer-speed: 0.2s;
    --uizy-drawer-easing: cubic-bezier(0.4, 0, 0.2, 1);

    /* Overlay */
    --uizy-overlay-color: black;
    --uizy-overlay-opacity: 0.45;
    --uizy-overlay-layer: 10;
}
```

## Complete Example

Here's a full working example with JavaScript:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <title>Uizy App</title>
    <meta name="viewport" content="width=device-width, initial-scale=1" />
</head>
<body>
    <uizy-app id="app">
        <uizy-system-bar>
            My App v1.0
        </uizy-system-bar>

        <uizy-header shadow="12">
            <button id="menu-btn">Menu</button>
            <span>Dashboard</span>
            <button id="settings-btn">Settings</button>
        </uizy-header>

        <uizy-drawer open shadow="6" clip-top clip-bottom>
            <nav>
                <a href="#">Home</a>
                <a href="#">Projects</a>
                <a href="#">Settings</a>
            </nav>
        </uizy-drawer>

        <uizy-drawer right shadow="6" clip-top clip-bottom>
            <h3>Details</h3>
            <p>Additional info here...</p>
        </uizy-drawer>

        <uizy-overlay clip-top clip-bottom></uizy-overlay>

        <uizy-main clip-top clip-bottom clip-left clip-right>
            <div class="pa-4">
                <h1>Welcome!</h1>
                <p>Your main content goes here.</p>
            </div>
        </uizy-main>

        <uizy-footer shadow="12">
            © 2024 My Application
        </uizy-footer>
    </uizy-app>

    <script type="module">
        import uizy from "@dufeut/uizy";

        uizy.layout({
            layout: {
                system: 24,
                header: 56,
                footer: 48,
                left: 240,
                right: 280,
                drawerSpeed: 0.2,
            },
        });

        uizy.init(() => {
            const app = uizy.get("app");
            if (!app) return;

            // Close overlay initially
            app.action("overlay", ({ set }) => set(false));

            // Menu button toggles left drawer
            document.getElementById("menu-btn").onclick = () => {
                app.action("left", ({ set }) => set());
            };

            // Settings button toggles right drawer
            document.getElementById("settings-btn").onclick = () => {
                app.action("right", ({ set }) => set());
            };
        });
    </script>
</body>
</html>
```

## Tips & Best Practices

:::tip Use Clip Attributes Wisely
- Use `clip-top` and `clip-bottom` on drawers to fit them between the header and footer
- Use `clip-system` when you want the drawer to extend behind the header
- Always use all four clip attributes on `<uizy-main>` for proper spacing
:::

:::tip Shadow Values
Shadow values range from 1-24. Common choices:
- `6` - Subtle elevation for sidebars
- `12` - Medium elevation for headers/footers
- `24` - High elevation for modals
:::

:::tip Responsive Design
The layout system automatically adjusts for mobile devices. Horizontal clips reset on smaller screens, and you can control visibility with breakpoint settings.
:::
