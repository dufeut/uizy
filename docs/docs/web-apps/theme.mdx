# Theme Configuration

Configure colors, scrollbar styles, and brand palettes.

## Overview

The theme system generates CSS custom properties for consistent styling across your application.

```javascript
uizy.start({
    theme: {
        colors: {
            primary: "#6b08a5",
            secondary: "#f5f5f5",
            accent: "#1eadff",
        },
        scrollbar: {
            size: 12,
            color: "rgba(121, 121, 121, 0.4)",
            hover: "rgba(121, 121, 121, 0.7)",
        },
        brands: [
            { name: "success", back: "#28b77b", text: "#fff" },
            { name: "danger", back: "#d64545", text: "#fff" },
        ],
    },
});
```

## Configuration Options

### System Colors

Define your color palette:

```javascript
uizy.theme.system({
    primary: "#6b08a5",
    secondary: "#f5f5f5",
    accent: "#1eadff",
    focus: "#1eadff",
    info: "#0050b9",
    success: "#28b77b",
    warning: "#f2c94c",
    danger: "#d64545",
    error: "#ff4d4d",
});
```

This generates CSS variables:

```css
:root {
    --color-primary: #6b08a5;
    --color-secondary: #f5f5f5;
    --color-accent: #1eadff;
    /* ... */
}
```

### Scrollbar

Customize scrollbar appearance:

```javascript
uizy.theme.scrollbar({
    size: 12,                           // Width in pixels
    color: "rgba(121, 121, 121, 0.4)",  // Thumb color
    hover: "rgba(121, 121, 121, 0.7)",  // Thumb hover color
});
```

Generates:

```css
:root {
    --scrollbar-size: 12px;
    --scrollbar-thumb: rgba(121, 121, 121, 0.4);
    --scrollbar-thumb-hover: rgba(121, 121, 121, 0.7);
}
```

### Brand Colors

Create color sets for buttons, badges, alerts, etc:

```javascript
uizy.theme.brand({ name: "primary", back: "#6b08a5", text: "#fff" });
uizy.theme.brand({ name: "success", back: "#28b77b", text: "#fff" });
uizy.theme.brand({ name: "danger", back: "#d64545", text: "#fff", line: "#b83030" });
```

Generates:

```css
:root {
    --color-primary-back: #6b08a5;
    --color-primary-text: #fff;

    --color-success-back: #28b77b;
    --color-success-text: #fff;

    --color-danger-back: #d64545;
    --color-danger-text: #fff;
    --color-danger-line: #b83030;
}
```

## API Reference

### `uizy.theme.system(colors)`

Set system-wide color variables.

| Parameter | Type | Description |
|-----------|------|-------------|
| `colors` | `Record<string, string>` | Color name to value mapping |

### `uizy.theme.scrollbar(options?)`

Configure scrollbar appearance.

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `size` | `number` | `22` | Scrollbar width in pixels |
| `color` | `string` | `rgba(121, 121, 121, 0.4)` | Thumb color |
| `hover` | `string` | `rgba(121, 121, 121, 0.7)` | Thumb hover color |

### `uizy.theme.brand(options)`

Add a brand color set.

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `name` | `string` | Yes | Brand name (used in variable names) |
| `back` | `string` | No | Background color |
| `text` | `string` | No | Text/foreground color |
| `line` | `string` | No | Border/line color |

### `uizy.theme.reset()`

Clear all brand styles (keeps system and scrollbar).

### `uizy.theme.clear()`

Clear all theme settings.

### `uizy.theme.toCSS()`

Get all theme CSS as a string.

```javascript
const css = uizy.theme.toCSS();
// "--color-primary: #6b08a5; --color-secondary: #f5f5f5; ..."
```

## Using via `start()`

Configure theme through the unified `start()` method:

```javascript
uizy.start({
    theme: {
        colors: {
            primary: "#6b08a5",
            secondary: "#f5f5f5",
        },
        scrollbar: {
            size: 10,
        },
        brands: [
            { name: "primary", back: "#6b08a5", text: "#fff" },
            { name: "success", back: "#28b77b", text: "#fff" },
        ],
    },
});
```

## Examples

### Dark Theme

```javascript
uizy.theme.system({
    background: "#1a1a1a",
    surface: "#2d2d2d",
    text: "#ffffff",
    textMuted: "#a0a0a0",
    primary: "#8b5cf6",
    accent: "#22d3ee",
});
```

### Using Colors in CSS

```css
.card {
    background: var(--color-surface);
    color: var(--color-text);
    border: 1px solid var(--color-primary);
}

.button-primary {
    background: var(--color-primary-back);
    color: var(--color-primary-text);
}

.alert-danger {
    background: var(--color-danger-back);
    color: var(--color-danger-text);
    border-color: var(--color-danger-line);
}
```

### Dynamic Theme Switching

```javascript
const themes = {
    light: {
        background: "#ffffff",
        surface: "#f5f5f5",
        text: "#1a1a1a",
        primary: "#6b08a5",
    },
    dark: {
        background: "#1a1a1a",
        surface: "#2d2d2d",
        text: "#ffffff",
        primary: "#a855f7",
    },
};

function setTheme(name) {
    uizy.theme.system(themes[name]);
    document.documentElement.dataset.theme = name;
}

setTheme("dark");
```

### Complete Theme Setup

```javascript
uizy.start({
    theme: {
        // Base colors
        colors: {
            // Backgrounds
            background: "#ffffff",
            surface: "#f8f9fa",

            // Text
            text: "#212529",
            textMuted: "#6c757d",

            // Brand
            primary: "#0d6efd",
            secondary: "#6c757d",
            accent: "#0dcaf0",

            // Status
            success: "#198754",
            warning: "#ffc107",
            danger: "#dc3545",
            info: "#0dcaf0",

            // Borders
            border: "#dee2e6",
            borderLight: "#e9ecef",
        },

        // Scrollbar
        scrollbar: {
            size: 8,
            color: "#dee2e6",
            hover: "#adb5bd",
        },

        // Brand palettes
        brands: [
            { name: "primary", back: "#0d6efd", text: "#fff" },
            { name: "secondary", back: "#6c757d", text: "#fff" },
            { name: "success", back: "#198754", text: "#fff" },
            { name: "danger", back: "#dc3545", text: "#fff" },
            { name: "warning", back: "#ffc107", text: "#000" },
            { name: "info", back: "#0dcaf0", text: "#000" },
        ],
    },
});
```
